# AILingo

A full-stack application with a Django REST API backend and a react frontend.

This application is a simple example of how AI can be used to tutor! Come up with questions, and evaluate answers.


## Project Structure

- `api/` - Django REST API backend
- `web/` - Frontend web application
- `terraform/` - Infrastructure as code

## Getting Started

### API

Navigate to the `api/` directory and follow the setup instructions there.

### Web

Navigate to the `web/` directory and follow the setup instructions there.

## Development

The project includes CI/CD workflows for automated testing and Docker image building.

## Deployment
The terraform directory also contains important information on how to deploy this application to google cloud.  
Simple migration work should allow for automating migration to Azure or AWS (or other platforms) as well.
