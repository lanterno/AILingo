# Schol√© Bubble Chart Web App

React application with Chart.js for displaying and interacting with bubble charts.

## Setup

```bash
# Install dependencies
bun install

# Start development server
bun run dev
```

The app will be available at `http://localhost:3000`

## Build

```bash
bun run build
```

## Features

- Interactive bubble chart using Chart.js
- Draggable bubbles that auto-save on release
- Elegant, modern UI
- Real-time updates from the API
